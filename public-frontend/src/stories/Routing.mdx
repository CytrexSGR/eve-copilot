# Routing

All routes are defined in `src/App.tsx`. Pages are lazy-loaded using `React.lazy()` with `Suspense` and a skeleton `PageLoader` fallback.

## Public Routes (No Auth Required)

These routes are accessible without EVE SSO authentication:

| Path | Component | Description |
|------|-----------|-------------|
| `/auth/callback` | `AuthCallback` | EVE SSO OAuth callback handler |
| `/impressum` | `Impressum` | Legal imprint (wrapped in Layout) |
| `/datenschutz` | `Datenschutz` | Privacy policy (wrapped in Layout) |

## Authenticated Routes

All remaining routes are wrapped in `RequireAuth`, which checks `useAuth().isLoggedIn` and displays a login prompt if the user is not authenticated. Authenticated routes are wrapped in `Layout` which provides the sidebar navigation and header.
